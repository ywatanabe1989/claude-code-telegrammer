Installation
============

From PyPI
---------

.. code-block:: bash

    pip install claude-code-telegrammer

From Source
-----------

.. code-block:: bash

    git clone https://github.com/ywatanabe1989/claude-code-telegrammer.git
    cd claude-code-telegrammer
    pip install -e .

Requirements
------------

- Python >= 3.10
- Claude Code CLI installed
- Telegram Bot Token (via BotFather)
